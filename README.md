# MyPersonalWeatherApp
A sleek, modern weather application for Android built with Jetpack Compose

##Features & Functionality
- Current Weather Tab: where you search for cities and view real-time meteorological data.
- History Tab: A dedicated space to track previous searches and weather
- Intelligent Solar Logic: The app automatically detects if it is day or night in the searched city to adjust UI elements.
-Time zone Synchronization: Accurate local sunrise and sunset times calculated specifically for the searched location's UTC offset.

## Technical Stack

- UI Framework: Jetpack Compose (Material 3)
- Language: Kotlin
-Architecture: MVVM (Model-View-ViewModel) with StateFlow for reactive UI updates


## How to Run
1. Clone the repository  
   `git clone https://github.com/JMLHP/MyWeatherApp.git`
2. Open the project in **Android Studio**.
3. Sync Gradle.
4. Run the app on an emulator or device.
